# Banking_R.A.G_model
🏦 Bank RAG Assistant — A fully local RAG pipeline using PyMuPDF for PDF extraction, Sentence Transformers (all-MiniLM-L6-v2) for embeddings, FAISS for vector search, and Llama 3.2 (Ollama) for generation. Streamlit frontend with chat history, source tagging, and context viewer.
